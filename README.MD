# Attendance System  repo backend 

Backend for Attendance sys repo

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install syllabus.

```bash
pip install -r requirements.txt
./run.sh

```
Cheers! There you go...
